# StudyKit Frontend

The frontend for StudyKit, built with React, TypeScript, and Vite.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

## Running Tests

This project uses [Vitest](https://vitest.dev/) for unit testing.

To run the tests:

```bash
npm test
```

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
